# 🚀 START-beta: AI-экосистема для разработки

Привет! Сегодня расскажу о своем проекте START-beta - интегрированной среде разработки с искусственным интеллектом. Это не просто очередной IDE, а целая экосистема из двух приложений, которые работают вместе с нейросетью.

## 🏗️ Архитектура проекта

Проект состоит из двух основных компонентов:

**StartIDE** - среда разработки с AI-ассистентом:
- Редактор кода с подсветкой синтаксиса
- Проводник файлов и папок  
- Интеграция с Ollama для получения AI-советов
- Автоматическая отправка контекста проекта в нейросеть

**Start Office** - офисное приложение для управления проектами:
- Обзор проектов и их структуры
- AI ассистент для вопросов о проектах
- История изменений в проектах
- Синхронизация с проектами StartIDE

## 🧠 Как работает AI-интеграция

Самое интересное - это общая нейросеть, которая владеет информацией о проекте. Вот как это реализовано:

### 1. Сбор контекста проекта
Каждый проект автоматически анализируется:
```python
# Структура проекта
📁 project_folder/
📄 main.py
📄 config.txt
📄 README.md

# Фрагменты кода (до 5000 символов на файл)
# История изменений
```

### 2. Форматирование для AI
Контекст преобразуется в удобный текстовый формат:
```
=== ПРОЕКТ: my_app ===
Путь: /path/to/project

=== СТРУКТУРА ПРОЕКТА ===
📁 src/
📄 main.py
📄 utils.py

=== ФРАГМЕНТЫ КОДА ===
Файл: main.py
--------------------------------------------------
import tkinter as tk
from tkinter import ttk
# ... код

=== ИСТОРИЯ ИЗМЕНЕНИЙ ===
- 2024-01-15: Создан файл main.py
- 2024-01-16: Добавлена функция calculate()
```

### 3. Интеграция с Ollama
```python
class OllamaManager:
    def __init__(self, base_url="http://localhost:11434", model="llama3.1"):
        self.base_url = base_url
        self.model = model
        
    def send_project_context(self, project_path, context_data):
        # Создаем .txt файл с контекстом
        context_text = self._format_context_for_ai(context_data, project_path)
        
        # Отправляем в нейросеть
        response = self.session.post(f"{self.base_url}/api/generate", 
            json={
                "model": self.model,
                "prompt": f"Изучи контекст проекта:\n\n{context_text}\n\nГотов к вопросам о проекте.",
                "stream": False
            }
        )
```

## 🔄 Обмен данными между приложениями

StartIDE и Start Office обмениваются данными через:
- `.project_context.json` - структурированный контекст проекта
- `.ai_context.txt` - отформатированный текст для нейросети
- Общие модули в папке `shared/`
