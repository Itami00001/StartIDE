# START-beta - Интегрированная среда разработки с AI

## Описание

START-beta - это экосистема из двух приложений:
- **StartIDE** - интегрированная среда разработки с поддержкой нейросети
- **Start Office** - офисное приложение для управления проектами с AI ассистентом

## Основные возможности

### StartIDE
- Редактор кода с подсветкой синтаксиса
- Проводник файлов и папок
- Интеграция с Ollama для получения AI советов
- Автоматическая отправка контекста проекта в нейросеть
- Создание и управление проектами в указанной директории

### Start Office
- Обзор проектов и их структуры
- AI ассистент для вопросов о проектах
- История изменений в проектах
- Синхронизация с проектами StartIDE

## Системные требования

- Python 3.7+
- Ollama (установленный и запущенный)
- Tkinter (обычно включен в Python)

## Установка и запуск

1. Клонирование репозитория:
```bash
git clone <repository-url>
cd START-beta
```

2. Установка зависимостей:
```bash
pip install -r requirements.txt
```

3. Установка и запуск Ollama:
```bash
# Установка Ollama (если еще не установлен)
curl -fsSL https://ollama.ai/install.sh | sh

# Запуск Ollama
ollama serve

# Загрузка модели (рекомендуется llama2)
ollama pull llama2
```

4. Запуск приложений:

**StartIDE:**
```bash
python StartIDE/main.py
```

**Start Office:**
```bash
python StartOffice/main.py
```

## Как это работает

### Интеграция с Ollama

Приложения используют Ollama API для взаимодействия с нейросетью. Контекст проектов отправляется в виде текстовых файлов (.txt) для лучшего понимания нейросетью.

### Обмен данными

- StartIDE автоматически отслеживает изменения в проектах
- Контекст сохраняется в файлах `.project_context.json`
- Start Office может читать этот контекст и отвечать на вопросы о проектах
- Нейросеть получает структуру проекта и фрагменты кода для анализа

### Формат контекста

Контекст для нейросети форматируется в удобный текстовый формат:
```
=== ПРОЕКТ: project_name ===
Путь: /path/to/project

=== СТРУКТУРА ПРОЕКТА ===
📁 folder1/
📄 main.py
📄 config.txt

=== ФРАГМЕНТЫ КОДА ===
Файл: main.py
--------------------------------------------------
[код файла]

=== ИСТОРИЯ ИЗМЕНЕНИЙ ===
- [записи об изменениях]
```

## Использование

1. **Создание проекта в StartIDE:**
   - Запустите StartIDE
   - Выберите "Файл" → "Новый проект"
   - Укажите папку для проекта
   - Начните создавать файлы и писать код

2. **Использование AI в StartIDE:**
   - Откройте панель "Нейро" (справа)
   - Задайте вопрос о вашем проекте
   - Нейросеть даст рекомендации на основе контекста

3. **Работа в Start Office:**
   - Запустите Start Office
   - Выберите проект из списка
   - Задавайте вопросы AI ассистенту о проекте
   - Просматривайте структуру и историю изменений

## Требования к Ollama

- Ollama должен быть запущен на `http://localhost:11434`
- Рекомендуемая модель: `llama2`
- Для установки других моделей используйте `ollama pull <model_name>`

## Возможные проблемы

1. **Ошибка подключения к Ollama:**
   - Убедитесь, что Ollama запущен
   - Проверьте, что порт 11434 доступен

2. **Проект не найден в Start Office:**
   - Убедитесь, что проект был создан в StartIDE
   - Проверьте наличие файла `.project_context.json` в папке проекта

3. **Нейросеть не отвечает:**
   - Проверьте подключение к Ollama
   - Убедитесь, что модель загружена

## Разработка

Проект написан на Python с использованием Tkinter для GUI. Архитектура построена на модульном подходе с общими компонентами в папке `shared/`.
